## **待工狀態頁面**

**API 路徑：**
`[GET]/api/StatusConfigurations/CheckPorterWaits`
### 資料格式 中英對照表

| 英文 (English)               | 中文 (Chinese)       |
|------------------------------|----------------------|
| PorterUID             | 傳送員唯一標識       |
| Block                        | 棟別名稱             |
| Level                        | 樓層名稱             |
| Account                      | 員工編號             |
| Name                         | 傳送員Name          |
| LastActionTime             | 待命開始時間         |
| IdleTime                    | 待工時間             |
| RestTimeFrom               | 休息開始             |
| RestTimeTo                 | 休息結束             |
| ReqType                 | 傳送對象             |
| IsOnDuty                   | 是否在職             |
| onDutyCount                   | 總上工人數             |
| onWaitCount                   |       待工中人數       |
| excutingCount                   |  執行中人數            |
